
# Project-Athena

## Description

A static website for a private aerospace company seeking to raise awareness of their brand and goals.

The webiste has two primary goals

- Communicate the business goals, history and culture.
- Provide a point of entry for intrested users looking to work with Project Athena.

Built utilising HTML, CSS and Javascript to present a professional, responsive and modern experience.

## Key Features

- Home page detailing the goals of the company.
- About page giving a visual history of the comapny.
- Join page allowing intrested users to express their intrest in joining the company.
- Contact page allowing visitors to submit their details and receive confirmation that their message was sent.
- Gallery page featuring a number of images that align with the company mission statement.
- Launch page for displaying future and past rocket launch videos.
- Careers page for viewing possible job vacancies.
- Responsive navigation.
- Footer sectionwith relevant links and copyright.
- Responsive design, allowing usage on all device screen sizes
- Consistent styling across the site.
